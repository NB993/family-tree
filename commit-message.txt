docs [with-ai] FT-017 Family 목록 및 검색 기능 통합 개발 문서 작성

- Phase 1과 Phase 2를 모두 포함한 FT-017 전체 개발 문서 작성 완료
- Epic 정보를 FT-014 Family 관리 시스템으로 올바르게 수정
- 최근 두 커밋(Phase 1, Phase 2) 내용을 종합하여 통합 문서화

## 작성된 문서 주요 내용

### 개발 개요
- Phase 1: 내 소속 Family 목록 조회 (GET /api/families/my)
- Phase 2: 공개 Family 검색 + 무한 스크롤 (GET /api/families/public)
- 헥사고날 아키텍처 기반 시스템 설계 및 구현 내용

### 기술적 구현 상세
- 애플리케이션 계층: FindMyFamiliesQuery, FindPublicFamiliesQuery, CursorPage 도메인 모델
- 인프라 계층: 커서 기반 페이징, CursorUtils, 데이터베이스 쿼리 최적화
- 프레젠테이션 계층: 경량화된 응답 DTO, 모바일 친화적 API 설계

### 품질 보증
- 단위 테스트 및 통합 테스트 전략과 결과
- 90% 이상 코드 커버리지 달성
- 성능 최적화 및 보안 고려사항 포함

### 시니어 개발자 관점
- 아키텍처 품질 분석 및 개선 포인트
- 확장성 고려사항 및 향후 발전 방향
- 운영 환경 대비 체크리스트 완료

## 문서 특징
- 실제 구현된 코드 기반의 정확한 문서화
- 향후 유지보수 및 확장을 위한 완전한 레퍼런스 자료
- API 명세서, 테스트 전략, 성능 고려사항 등 운영 준비 완료
