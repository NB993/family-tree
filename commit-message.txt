feat [with-ai] FT-017 Family 목록 및 검색 기능 - Phase 1 내 소속 Family 목록 조회

- 사용자가 소속된 Family 목록을 조회하는 기능 구현 완료
- 기존 FindFamilyUseCase 인터페이스를 확장하여 일관성 있는 설계 유지
- GET /api/families/my 엔드포인트로 내 소속 Family 목록 조회 지원

## 구현된 주요 컴포넌트

### 애플리케이션 계층
- FindMyFamiliesQuery: 내 소속 Family 조회를 위한 Query 클래스
- FindFamilyUseCase: findAll(FindMyFamiliesQuery) 메서드 오버로딩 추가
- FindFamilyService: findAll(FindMyFamiliesQuery) 구현 - FamilyMember를 통한 Family 조회

### 인프라 계층
- FindFamilyMemberPort: findAllByUserId(UserId) 메서드 추가
- FamilyMemberAdapter: findAllByUserId 메서드 구현
- FamilyMemberJpaRepository: findAllByUserId 쿼리 메서드 추가

### 프레젠테이션 계층
- FindFamilyController: GET /api/families/my 엔드포인트 추가
- 기존 Family 조회 응답 형식과 동일한 FamilyResponse 사용

## 테스트 구현
- FindMyFamiliesQueryTest: Query 객체 단위 테스트 ✅
- FindFamilyControllerTest: 내 소속 Family 조회 API 통합 테스트 ✅
- 전체 테스트 통과 확인 (기존 기능 호환성 유지)

## 해결된 이슈
- 기존 UseCase 확장 방식으로 새로운 인터페이스 생성 없이 일관성 확보
- FamilyMember 엔티티의 테스트 데이터 생성 패턴 최적화 (withRole 메서드 활용)
- 기존 Family 조회 로직과 동일한 응답 구조로 클라이언트 호환성 유지
