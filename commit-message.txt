## 구현된 주요 컴포넌트

### 웹 계층 (Controller)
- TokenController: JWT 토큰 갱신/로그아웃 API 엔드포인트
  - POST /api/auth/refresh: Refresh Token을 사용한 토큰 갱신
  - POST /api/auth/logout: 인증된 사용자 로그아웃 처리

### DTO 계층
- TokenRefreshRequest: 토큰 갱신 요청 DTO (검증 어노테이션 포함)
- LogoutResponse: 로그아웃 응답 DTO (성공/실패 팩토리 메서드)

## API 명세

### POST /api/auth/refresh
- 요청: { "refreshToken": "eyJ..." }
- 응답: { "accessToken": "eyJ...", "refreshToken": "eyJ...", "tokenType": "Bearer", "expiresIn": 300 }

### POST /api/auth/logout  
- 요청: Authorization Header 필요
- 응답: { "success": true, "message": "로그아웃이 성공적으로 완료되었습니다." }
