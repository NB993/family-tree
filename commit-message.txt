## 해결된 이슈

### 테스트 실패 원인 분석 및 해결
- 문제: SaveFamilyControllerTest에서 201 응답 대신 USER_NOT_FOUND 예외 발생
- 원인: @WithMockOAuth2User는 SecurityContext만 설정하고 실제 DB에 User 없음
- 해결: 각 테스트에서 실제 User 생성 후 FTUser Principal로 인증 처리

### 테스트 격리 개선
- ObjectOptimisticLockingFailureException 해결을 위해 User.newUser() 사용
- 자동 생성된 User ID를 사용하는 FTUser Principal 생성
- @AfterEach에서 관련 데이터 완전 정리 (FamilyMember, Family, User 순)
