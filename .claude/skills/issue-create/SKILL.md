# GitHub 이슈 생성

## 개요

Claude와 논의한 작업 계획(신규 개발, 리팩토링, 버그 수정 등)을 GitHub 이슈로 등록합니다.

## 워크플로우

1. **대화 내용 분석**: 논의된 내용에서 이슈 제목과 본문 도출
2. **이슈 타입 결정**: 내용에 따라 적절한 라벨 선택
3. **이슈 생성**: gh CLI로 이슈 생성
4. **라벨 추가**: 이슈 타입 + 작업 영역 라벨 추가
5. **프로젝트 연결**: olhaeolhae 프로젝트에 연결
6. **마일스톤 지정**: 사용자에게 확인 후 지정

## 금지사항

- 대화 내용 없이 임의로 이슈 생성 금지
- 중복 이슈 생성 주의
